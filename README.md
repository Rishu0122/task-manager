# 📝 Task Manager

A full-stack Task Manager application built with Node.js, Express.js, MongoDB, and Vanilla JavaScript. Users can register, log in securely using JWT authentication, and manage their personal tasks.

## 🚀 Features

- 🔐 User Authentication (Register & Login)
- 👤 User Profile
- ➕ Create Task
- 📋 View All Tasks
- ✏️ Update Task
- ❌ Delete Task
- 🔒 JWT Authentication
- 🛡️ Ownership Authorization (Users can only manage their own tasks)
- 📱 Responsive UI

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Rishu0122/task-manager.git
```

### Backend

```bash
cd task-manager/task-manager-backend
npm install
npm start
```

### Frontend

Open the frontend files using Live Server or any static web server.

---

## 🔑 Environment Variables

Create a `.env` file inside `task-manager-backend`.

```
PORT=3000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
```
